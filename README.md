# Water quality Analysis & Forecast
*The project:* Interactive dashboard for analyzing and predicting water quality in different cities.

## Demo

![App Screenshot](screenshots streamlit)
## Description
The project allows you to:
- Upload CSV with water quality data
- Select one or more cities for analysis
- Build interactive graphs with metrics: pH, turbidity, conductivity
- Make a forecast for future days using linear regression

## Programms
- Python 3
- Streamlit for the web interface
- Plotly for interactive visualization
- Pandas and scikit-learn for data processing and forecasting

## How to launch a project
1. Clone the repository:
   git clone https://github.com/soluna-moon/waterquality-forecast-data
2. Install the requirements:
   pip install -r requirements.txt
3. Launch Streamlit:
   streamlit run waterquality_data.py
